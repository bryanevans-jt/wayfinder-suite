# Joshua Tree Reports — Georgia guide

**Formal reporting · Google Drive PDFs**

App URL: **wayfinder-reports.thejoshuatree.org**

---

## 1. Purpose

Joshua Tree Reports is separate from Wayfinder Pro day-to-day casework. It generates **official PDFs**, emails copies, and saves to **Google Drive** folders configured by administration.

---

---

## 5. SE Monthly (most common)

1. Select **Georgia** state.
2. Choose **SE Monthly**.
3. Select **client** from your caseload (synced from Wayfinder).
4. Review **prefill** from contact logs (job development section).
5. Complete remaining fields; capture **signature** when required.
6. Submit → PDF generated → email + Drive.

## 6. Recall / prior month data

Reports may **recall** fields from prior SE Monthly submissions for the same client — review before submitting.

- **Missing reports** cron — alerts for overdue SE Monthly.

## 8. Troubleshooting

| Issue | Action |
|-------|--------|
| Client not in dropdown | Confirm client on your ES caseload in Pro; GA office |
| Prefill empty | Log contacts in Pro first; check reporting month |
| Google / PDF error | Note WF- code; email Bryan |
| Wrong state | GA training uses GA only — TN is separate session |
